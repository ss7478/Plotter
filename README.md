# ЕТЗ - Единое Техническое Задание для ВсОШ по информатике 2026, или робот-плоттер для рисования на вертикальной поверхности.

Робот на основе контроллера Arduino Mega 2560. Перемещается по полигону, представляющему из себя вертикальную пластину из ДСП 16 мм, с закрепленными по бокам зубчатыми рейками.
Устройство перемещается по полигону по вертикали с помощью четырех шаговых двигателей Nema 17; по горизонтали перемещается только пишущая головка с маркером с помощью кинематики на двух линейных валах и ремня ГРМ.
На роботе имеется интерфейс для выбора исполняемой программы с помощью кнопки переключения задач, кнопки старта и семисегментного индикатора.

## Электроника и оборудование
- Arduino Mega 2560 - главный контроллер
- A4988*5 - драйверы шаговых двигателей.
- Шаговые двигатели Nema 17*5 - для перемещения устройства
- Сервопривод MG90S - для подъема и опускания пишущей головки с маркером.
- Датчики линии на TCRT5000 - для обнаружения черной линии, ограничивающей рабочую зону
- Концевик - для калибровки пишущей головки по оси X

Для размещения и подключения всех электронных компонентов в САПР EasyEDA разработана материнская плата. 
Она изготовлена вручную по ЛУТ-технологии. Трассировка платы, общая схема Э3 робота и структурная схема Э1 по ГОСТ представлены в папке docs/schematic.
